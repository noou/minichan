# Minichan - Анонимный имиджборд

Минималистичный анонимный имиджборд, созданный на FastAPI с бруталистским дизайном.

## Особенности

- **Анонимность**: Никаких аккаунтов или регистрации
- **Треды**: Создание тем с заголовком, текстом и изображением
- **Ответы**: Возможность отвечать в тредах с текстом и изображениями
- **Бруталистский дизайн**: Минималистичный интерфейс в стиле 4chan
- **Простота**: Легковесное решение на SQLite

## Технологии

- **Бэкенд**: Python, FastAPI
- **База данных**: SQLite
- **Фронтенд**: Jinja2 + Tailwind CSS (CDN)
- **Хранение изображений**: Локальная папка `/uploads`

## Установка и запуск

### 1. Установите зависимости

```bash
pip install -r requirements.txt
```

### 2. Запустите сервер

```bash
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

Или просто:

```bash
python main.py
```

### 3. Откройте в браузере

Перейдите по адресу: http://localhost:8000

## Структура проекта

```
minichan/
├── main.py              # Сервер и логика приложения
├── database.py          # Модели данных и настройка БД
├── requirements.txt     # Зависимости Python
├── templates/
│   ├── index.html       # Главная страница (список тредов)
│   └── thread.html      # Страница треда (ответы)
└── uploads/             # Папка для загруженных изображений
```

## Использование

1. **Создание треда**: На главной странице заполните форму (заголовок, текст, изображение)
2. **Просмотр треда**: Кликните на заголовок треда или кнопку "ОТВЕТИТЬ"
3. **Ответ в треде**: Используйте форму внизу страницы треда
4. **Удаление**: Кнопка [X] для удаления поста/треда (с подтверждением)

## API Endpoints

- `GET /` - Главная страница со списком тредов
- `POST /create` - Создание нового треда
- `GET /thread/{thread_id}` - Просмотр треда
- `POST /thread/{thread_id}/reply` - Ответ в треде
- `POST /thread/{thread_id}/delete` - Удаление треда
- `POST /post/{post_id}/delete` - Удаление поста

## Ограничения

- Максимальный размер файла: 5MB
- Поддерживаемые форматы: JPG, JPEG, PNG, GIF, WEBP
- Максимальная длина заголовка: 200 символов
